# 3D Space Portfolio

An interactive 3D portfolio website built with React Three Fiber.

## Features

- 🪐 3D Planets representing projects
- 🚀 Animated Spaceship navigation
- ☄️ Interactive Comet with blast effect
- 💓 Heartbeat pulse animation on selected planets
- 📁 Project Archive with category filter
- 🎵 Sound effects for interactions

## Tech Stack

- React 19
- Three.js / React Three Fiber
- Tailwind CSS
- Vite
- TypeScript

## Run Locally

```bash
npm install
npm run dev
